How to use a pack
1. Open the session_packs.md file. Find today's pack by ID (e.g. M4, E3, CS-P1.2).
2. Copy everything from the AI INSTRUCTION BLOCK down to the SESSION REPORT template at the bottom.
3. Paste into Gemini (or any AI). The instruction block turns it into a strict examiner — no hints, cold recall only.
4. Work through Tier 1 → 4. If you score 0 on Tier 1, the remediation block activates. No going back on Tier 4.
5. Fill in the session report. Log your confidence here via the Log Session button.

Dashboard Guide
How everything works.
Session structure (post-Easter)

Free days (Tue/Fri): 2 heavy sessions — Maths 11am–1pm hard cap (2hrs), CS or Econ 11am–12:30pm (1.5hrs).

College days (Mon/Wed/Thu): 1 light evening session only (~45min–1hr). Never stack two full sessions on a college day.

Weekends (Sat/Sun): 2 sessions per day, same structure as free days.

Hard cap: If Maths runs over 2hrs, the CS/Econ session is cancelled — not rescheduled to the same day. Burnout is the structural risk.

Missed sessions: Roll high-priority topics to next free slot. Drop low-priority topics if time runs short before an exam.
