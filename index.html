Dashboard Guide
How everything works, start to finish.
Countdown strip

The three panels at the top count down days to each subject's first exam. The progress bar fills as revision time passes since 9 March.

Today tab — ticking off sessions

Tick a session checkbox to mark it done. This immediately shows confidence buttons inline — click Seen it, Getting there, or Confident to log how well it went. This updates the Priority Gaps urgency score and the Focus card. You don't need to open the session log separately for tasks you tick off here.

28-day heatmap — each cell darkens as you log sessions. Hover for date and count. Amber border = today.

This Week tab

Mon–Sun grid for the current week. Today is highlighted amber, exam deadlines in red, weekends slightly dimmed.

All Sessions tab

Every session from today onwards, listed in date order. Use the filter buttons to narrow by subject. Exam days are highlighted in red.

Session Log (timer button)

Click Log session at any time to log topics without the timer. The 1-hour timer auto-opens the log when it completes.
